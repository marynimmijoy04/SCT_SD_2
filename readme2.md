Number Guessing Game

Description

The Number Guessing Game is a simple C++ console application where the computer generates a random number between 1 and 10. The player must guess the correct number, and the program provides hints indicating whether the guess is too high or too low until the correct number is found.

Features

* Random number generation between 1 and 10
* User-friendly console interface
* Hints for incorrect guesses
* Tracks the number of attempts
* Input validation for numbers outside the allowed range

Technologies Used

* C++
* Standard Library (iostream, cstdlib, ctime)

How to Run

1. Compile the program using a C++ compiler.
2. Run the executable.
3. Enter your guesses and try to find the correct number.

Learning Outcomes

This project helped in understanding:

* Random number generation
* Loops and conditional statements
* User input and output handling
* Basic problem-solving in C++
